# vue-form-making

基于 [vue2](https://v2.cn.vuejs.org/) 和 [element-ui](https://element.eleme.io/#/zh-CN) 实现的可视化表单设计器，让表单开发简单而高效。

### 特点

- 国际化，按需加载语言包
- 优雅美观

### 使用

1. 下载项目代码

```bash
git clone https://github.com/zheng-shugan/vue-form-making.git my-vue-form-making
```

2. 下载项目依赖

```bash
yarn install
```

或者

```bash
npm install
```

3. 启动本地服务器

```bash
yarn serve
```

或者

```bash
npm run serve
```
